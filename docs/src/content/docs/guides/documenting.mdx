## Quick Start

### Clone the repository

Open your terminal and run the following command:

```bash title="Terminal"
git clone https://github.com/Superp0sit1on/sonny.git
```

### Install the dependencies

On the terminal, navigate to the `docs` directory and run the following command:

<Tabs syncKey="package-managers">
  <TabItem label="npm" icon="seti:npm">
    ```bash title="Terminal"
    npm install
    ```
  </TabItem>
  <TabItem label="pnpm" icon="pnpm">
    ```bash title="Terminal"
    pnpm install
    ```
  </TabItem>
  <TabItem label="yarn" icon="seti:yarn">
    ```bash title="Terminal"
    yarn install
    ```
  </TabItem>
  <TabItem label="bun" icon="bun">
    ```bash title="Terminal"
    bun install
    ```
  </TabItem>
</Tabs>

### Run the local environment

On the terminal, run the following command:

<Tabs syncKey="package-managers">
  <TabItem label="npm" icon="seti:npm">
    ```bash title="Terminal"
    npm run dev
    ```
  </TabItem>
  <TabItem label="pnpm" icon="pnpm">
    ```bash title="Terminal"
    pnpm run dev
    ```
  </TabItem>
  <TabItem label="yarn" icon="seti:yarn">
    ```bash title="Terminal" 
    yarn run dev 
    ```
  </TabItem>
  <TabItem label="bun" icon="bun">
    ```bash title="Terminal" 
    bun run dev 
    ```
  </TabItem>
</Tabs>

### Edit the documentation

Locate the `.mdx` page file you want to edit in the `docs/src/content/docs` directory and make your changes.

:::note
If you are editing the `getting-started.mdx` or the `guides/contributing.mdx` files, don't forget to also update the `README.md` or `CONTRIBUTING.md` files of the project.
:::
