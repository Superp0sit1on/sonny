## Quickstart

### Create a new app

1. Log in to your [Heroku account](https://dashboard.heroku.com/apps), and create a new app.

### Setup your app
3. Go to the **Settings** tab.
    - On the **Config Vars** section, set the environment variables.
    - On the **Buildpacks** section, add the Python buildpack.
    - On the **SSL Certificates** section, set up the SSL Certificate.
4. Go to the **Deploy** tab.
    - On the **Deployment method** section, select your preferred deployment method.

### Activate your Dyno

5. Go to the **Resources** tab.
    - Activate your Dyno.
6. Check your chatbot live on Twitch.
